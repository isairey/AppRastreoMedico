# 🏥📡 Sistema de Rastreo Médico

Aplicación web desarrollada para el **seguimiento y monitoreo de pacientes**, permitiendo gestionar ubicaciones, estados de salud y eventos médicos en tiempo real.

---

## 📌 Descripción

El **Sistema de Rastreo Médico** es una solución orientada al control y supervisión de pacientes dentro de hospitales, clínicas o entornos de atención remota.

La plataforma permite registrar información médica relevante y, en conjunto con sistemas de geolocalización o monitoreo, facilita el seguimiento continuo del paciente.

---

## 🚀 Características

- 🧑‍⚕️ Registro y gestión de pacientes  
- 📍 Rastreo de ubicación (opcional)  
- ❤️ Monitoreo de estado de salud  
- 🚨 Alertas médicas  
- 📊 Historial de eventos clínicos  
- 🔐 Autenticación de usuarios  
- ⚡ Panel de control  

---

## 🛠️ Tecnologías utilizadas

### Backend
- **Java** → Lógica del sistema  

### Frontend
- **HTML5** → Estructura  
- **CSS3** → Estilos

- **Java Servlets / JSP** → Integración web
